# 📱 PixelDroid Website

A sleek, high-performance, and responsive landing page for the **PixelDroid GSI**, specifically optimized for the **Samsung Galaxy A12 (Exynos 850)**.

---

## 🚀 Features

* **Glass-morphism UI:** Modern, clean "Pixel" aesthetic with blur effects and smooth transitions.
* **Dynamic Builds:** Fetch latest downloads automatically from `versions.txt` without editing HTML.
* **Fully Responsive:** Optimized for both mobile and desktop viewing.
* **Optimized Performance:** Minimal JS and CSS footprint for fast loading.
* **User-Friendly:** Interactive "Back to Top" button and scroll-reveal animations.
