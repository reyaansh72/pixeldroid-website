# pixeldroid-website

This repository contains the source code for the PixelDroid website.

📦 Clone the Repository
git clone https://github.com/YOUR_USERNAME/pixeldroid-website.git
📁 Navigate to Project
cd pixeldroid-website
▶️ Run Locally

Start a local server:

python -m http.server

Then open in your browser:

http://localhost:8000
🌐 GitHub Pages Deployment
Go to repository Settings
Open Pages
Set:
Source: Deploy from branch
Branch: main
Folder: / (root)
Save

Your site will be available at:

https://YOUR_USERNAME.github.io/pixeldroid-website
🔄 Updating the Website

To update the site:

git add .
git commit -m "Update site"
git push origin main
⚠️ Notes
Ensure index.html exists in the root directory
Check file paths before pushing
This project is a static website
Works best with a local server or GitHub Pages
